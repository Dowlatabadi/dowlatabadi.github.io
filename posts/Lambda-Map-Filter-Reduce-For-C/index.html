Lambda-Map-Filter-Reduce-For-C#!

Jan 29 2021-01-29T00:00:00+08:00 by Mehrdad Dowlatabadi
Updated Feb 20 2021-02-20T09:27:33+08:00 1 min

Quick look at equvalents of useful functionalities in c#:

C#Python
Func
x => x * x		Lambda function
lambda x: x * x
Select
l1.select(x=>x*x)		Map
list(map(lambda x:x * x,l1)) ☰ [x * x for x in l1]
Where
l1.Where(x=>x>10)		Filter
list(filter(lambda x:x>10,l1)) ☰ [x for x in l1 if x>10]
Aggregate
l1.Aggregate((x, y) =>x*y)		Reduce
reduce(lambda x,y:x*y,l1)
